虚拟机安装ubuntu桌面版

发表于 2021-06-20 更新于 2022-01-10 分类于 操作系统 阅读次数：

^ _ ^

下载 ubuntu ios

  1. 前往清华镜像站：https://mirrors.tuna.tsinghua.edu.cn/。
  2. 在搜索框中搜索 ubuntu 关键字。
  3. 进入 ubuntu-release 文件夹中，选择合适版本进行下载。我选择的是 21.04 版。

VMWare 安装 ubuntu

步骤链接：https://zhuanlan.zhihu.com/p/141033713

问题1：全屏显示

安装链接中有提到 全屏显示 问题的解决方法，这里对没有提及的一些要点进行补充：

  1. VMware Tools 并不是本来就在桌面上的，需要点击 虚拟机 –> 安装 VMware Tools 进行安装。
  2. 安装成功后，Ubuntu 桌面上会出现名为 VMware Tools 的文件夹。进入文件夹，解压后缀为 tar.gz 的压缩包。解压过程中，可能会出现 Not enough space to extract 错误提示。解决这个错误的方法是将压缩包复制到另外一个目录进行解压。
  3. 解压成功后会出现一个 vmware-tools-distrib 文件夹，在 terminal 中进入该文件夹，运行该文件夹下的 vmware-install.pl 脚本，执行命令为：

问题2：时间调整

安装后 ubuntu 系统显示的时间不是很准确，在 Settings 中的 Date & Time 下进行手动修改即可。

问题3：网络连接

方案一：编辑 –> 虚拟网络编辑器 –> 更改设置 –> 还原默认设置
