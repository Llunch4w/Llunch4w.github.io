使用方法

Step1 : 获取音乐的链接

打开网易云网站，找到希望添加的歌曲id，比如这首没有理想的人不伤心是1305364671。

Step2 : 获取封面图片的链接

  • 在图片处右键，然后检查图片
  • 找到 img 结点，获取它的 src 值，即为图片链接

标签参数：

  • title : 歌曲名称
  • author : 歌曲作者
  • url : Step1 获取到的歌曲外链url地址
  • picture_url（封面|可选） : Step2 获取到的图片url地址
  • narrow: （可选）播放器袖珍风格
  • autoplay: (可选) 自动播放，移动端浏览器暂时不支持此功能
  • width:xxx:(可选) 播放器宽度(默认: 100%)
